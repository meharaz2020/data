Question 1

Consider a scenario where you are managing a company‘s employee

database, and you have a table named ‘employees‘ with columns such as

‘employee_id,‘ ‘department_id,‘ and ‘salary.‘ Your task is to write a SQL query to

identify the top 3 salaries in each department.


Provide a query that selects the ‘department_id,‘ ‘employee_id,‘ and ‘salary‘ for

the top 3 salaries in each department. Use appropriate aliases for clarity in the

result set. Assume that the table is named ‘employees‘ and contains relevant

data.

Answer: 
            WITH RankedSalaries AS (

                    SELECT

                        department_id,

                        employee_id,

                        salary,

                        ROW_NUMBER() OVER (PARTITION BY department_id ORDER BY salary DESC) AS salary_rank

                    FROM

                        employees

            )

            SELECT

                    department_id,

                    employee_id,

                    salary

            FROM

                    RankedSalaries

            WHERE

                    salary_rank <= 3;

Question 2

You‘re managing a database for a small bookstore. The database contains two tables: ‘Books‘ and

‘Authors‘.

Tables:

Books:

Columns: BookID (Primary Key), Title, AuthorID (Foreign Key), Genre, Price

Authors:

Columns: AuthorID (Primary Key), AuthorName, Nationality


Write an SQL query that retrieves the title and price of all books along with the corresponding author‘s

name.

Answer: 
            SELECT

                    B.Title,

                    B.Price,

                    A.AuthorName

            FROM

                    Books B

            INNER JOIN

                    Authors A ON B.AuthorID = A.AuthorID;

Question 3

Considering a table named "Orders" with columns: "OrderID", "OrderDate", and "TotalAmount", what

query would find the percentage of the total amount for each order against the total amount of all

orders?

Answer: 
                SELECT

                        OrderID,

                        OrderDate,

                        TotalAmount,

                        (TotalAmount * 100.0) / SUM(TotalAmount) OVER () AS Percentage

                FROM

                        Orders;

Question 4

You‘re tasked with managing a database for an e-commerce website. The database contains several

tables, including ‘Products‘, ‘Customers‘, ‘Orders‘, and ‘OrderDetails‘. The company wants to retrieve

specific information about their top-selling products in the last quarter.


Create an SQL query that fetches the top 5 best-selling products based on the quantity sold within the

last quarter. Include the product name, total quantity sold, and revenue generated by each product.

Additionally, ensure that only products with sales in the last quarter are included in the result.


Tables:

Products:(ProductID, ProductName)

Orders:( OrderID, CustomerID, OrderDate)

OrderDetails:( OrderID, ProductID, Quantity, UnitPrice)


Your query should filter data for the last quarter based on the OrderDate column in the Orders table.

Answer: 
        WITH LastQuarterOrders AS (

                SELECT

                    OD.ProductID,

                    P.ProductName,

                    SUM(OD.Quantity) AS TotalQuantity,

                    SUM(OD.Quantity * OD.UnitPrice) AS Revenue

                FROM

                    OrderDetails OD

                INNER JOIN

                    Orders O ON OD.OrderID = O.OrderID

                INNER JOIN

                    Products P ON OD.ProductID = P.ProductID

                WHERE

                    O.OrderDate >= DATEADD(QUARTER, DATEDIFF(QUARTER, 0, GETDATE()) - 1, 0)

                    AND O.OrderDate < DATEADD(QUARTER, DATEDIFF(QUARTER, 0, GETDATE()), 0)

                GROUP BY

                    OD.ProductID, P.ProductName

        )

        SELECT TOP 5

            ProductID,

            ProductName,

            TotalQuantity,

            Revenue

        FROM

            LastQuarterOrders

        ORDER BY

            TotalQuantity DESC;

Question 5

Customers Table:

CustomerID (Primary Key),FirstName,LastName,BirthDate,City

Transactions Table:

TransactionID (Primary Key),CustomerID (Foreign Key),Product,Quantity,Price,PurchaseDate


Write an SQL query to identify customers who have purchased at least 2 quantities of products (chips

and chocolate), with the total price not exceeding 300 taka. Additionally, these customers must be:

Aged over 22 years old based on their birth date.Living in Dhaka.

The query should retrieve the names of customers who meet these criteria.

Answer: 
                WITH CustomerTransactions AS (

                        SELECT

                            t.CustomerID,

                            SUM(t.Quantity) AS TotalQuantity,

                            SUM(t.Price) AS TotalPrice

                        FROM

                            Transactions t

                        WHERE

                            t.Product IN ('chips', 'chocolate')

                        GROUP BY

                            t.CustomerID

                )

                SELECT

                    c.FirstName,

                    c.LastName

                FROM

                    Customers c

                JOIN

                    CustomerTransactions ct ON c.CustomerID = ct.CustomerID

                WHERE

                    c.City = 'Dhaka'

                    AND DATEDIFF(YEAR, c.BirthDate, GETDATE()) > 22

                    AND ct.TotalQuantity >= 2 

                    AND ct.TotalPrice <= 300;
